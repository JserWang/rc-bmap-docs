地图的平移缩放控件，可以对地图进行上下左右四个方向的平移和缩放操作。

## API

### 属性

| 属性  | 类型 | 默认取值 | 说明 |
|-------|-----|------|-----|
| anchor | `CONTROL_ANCHOR` | / | 控件的停靠位置，从`Constants`中引入 |
| offset | `Size`  | / | 控件的水平偏移值 |
| type | `NAVIGATION_CONTROL_TYPE`  | / | 控件的水平偏移值，从`Constants`中引入 |
| showZoomInfo | `Boolean`  | / | 是否显示级别提示信息 |
