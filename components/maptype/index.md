切换地图类型的控件

## API

### 属性

| 属性  | 类型 | 默认取值 | 说明 |
|-------|-----|------|-----|
| anchor | `CONTROL_ANCHOR` | / | 控件的停靠位置，从`Constants`中引入 |
| offset | `Size`  | / | 控件的水平偏移值 |
| type | `MAP_TYPE_CONTROL_TYPE`  | / | 控件样式，从`Constants`中引入 |
| mapTypes | `Array[MAP_TYPE]`  | / | 控件展示的地图类型，从`Constants`中引入 |
